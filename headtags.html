Head Tags and Meta Data

A guide for HTML meta data <head> tags.

This Head Tags and Meta Data site suggests HTML to apply to your document. The <head> tags are essentially a container for meta data which define attributes related to the document. This meta data is not displayed on the web page, rather it provides browsers and search engines specific information about the page for example: document title, description, links and scripts.
